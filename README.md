# Workshop Documentation

## Overview
This is a simple customercare chatbot for a fake company called **SunriseRecords**.
It allows users to ask questions about the company's products and services.

## Installations 
### All commands are run in the CLI
Majority of this project requires no installations, however Node.js is required.

### Download the Node.js Installer
Open your web browser and navigate to the official [Node.js](https://nodejs.org/en)

#### Run the Installer
Locate the downloaded installer file and double-click to run it.
Click "Next" on the welcome screen.
Follow the instructions and ensure "Add to PATH" is selected.
Skip any options and click "Install" to complete the installation process.
Once the installation is complete, click "Finish".

#### Verify the installation
Open your command prompt(Windows) or terminal (macOS/Linux).
Type the following commands and press "Enter":
```
node -v
npm -v
```

### Download and Install Git
Open your web browser and navigate to the official Git website.
Download the appropriate version for your operating system (Windows, macOS, or Linux).
Run the downloaded installer.
For most users, the default options during installation process are sufficient. Click "Next" throught the prompts until the installation is complete.

#### Verify the Git installation
Open your command prompt or terminal. Type the following command and press "Enter":
```
git --version
```

## Next Steps
Now that you have installed Node.js and Git, you will now clone this GitHub repo.

## Clone the Repo
Navigate to the desired directory where you want to store the project in your terminal or command prompt.
Run the following command:
```
git clone https://github.com/nimithejeagerist/chatbot.git
```
Once the cloning process is complete, navigate into the new project directory
```
cd chatbot
```
