## Installations
To create the React App, we will run:
```
npx create-next-app@latest demo-chatbot
```

Make sure to click **No, customize settings** when the pop-up appears, for this demo we will be using JavaScript as opposed to TypeScript.
These are the options that should be picked when customizing:
- Would you like to use TypeScript? ... No
- Which linter would you like to use? Â» ESLint
- Would you like to use React Compiler? ... No
- Would you like to use Tailwind CSS? ... Yes
- Would you like your code inside a `src/` directory? ... Yes
- Would you like to use App Router? (recommended) ... Yes
- Would you like to customize the import alias (`@/*` by default)? ... Yes
- What import alias would you like configured? ... @/*
